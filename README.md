# Machine_Translation
# Low-Resource Machine Translation (Arabic–Igbo)

## Overview
This project focuses on building a neural machine translation system for low-resource languages using Transformer-based models.

## Key Features
- Fine-tuned M2M100 multilingual model
- Custom preprocessing pipeline (cleaning, alignment, filtering)
- Evaluation using BLEU and chrF metrics

## Tech Stack
- Python, PyTorch, HuggingFace Transformers

## Results
- Significant improvement over zero-shot baseline
